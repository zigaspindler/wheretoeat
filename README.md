Where to eat?!
==============
App for simple voting where go to eat.

Supports
--------
- multiple restaurants
- daily offers
- permanent offers
- top 5 restaurants
- voting history
- chat
- fetch restaurant offers from [kam-jest](https://github.com/matjaz/kam-jest)
- user authentication
- [Shortreckonings](http://www.shortreckonings.com/)
